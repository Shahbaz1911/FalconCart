# **App Name**: Falcon Cart

## Core Features:

- Product Listing: Display product list with images, descriptions and prices in a grid.
- Product Detail Page: Show detailed product information with a larger image, detailed description, and 'Add to Cart' button.
- Cart Management: Add items to the cart, display current cart contents, and update the quantity of items.
- Checkout Process: Handle the checkout process, collecting shipping information and payment details (Stripe integration).
- AI-Powered Recommendations: Recommend related or complementary products to the current product using a tool.
- Order Tracking: Allow users to view their order history and current order status.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which translates to RGB hex #3399FF, to create a modern and trustworthy feel.
- Background color: HSL(210, 20%, 95%) which translates to RGB hex #F0F8FF, to keep things light and uncluttered.
- Accent color: HSL(180, 60%, 40%) which translates to RGB hex #33BDBD, to highlight key actions.
- Body font: 'Inter' sans-serif for a clean and readable user experience.
- Headline font: 'Space Grotesk' sans-serif for bolder headers, especially in product titles.
- Use clear, minimalist icons to represent categories, cart actions, and user actions.
- A clean and intuitive layout with clear separation of content sections.
